# Claude Code v2.1.45 リリース: Claude Sonnet 4.6 対応とスピナーヒントのカスタマイズ機能

## 概要

Claude Code v2.1.45 では、新モデル **Claude Sonnet 4.6** のサポートが追加され、スピナーのヒント表示をカスタマイズできる機能が実装されました。また、SDK 開発者向けにレート制限情報を取得するための新しい型が追加されるなど、開発者体験の向上に重点を置いたアップデートとなっています。バグ修正も複数含まれており、安定性の改善も図られています。

## 変更点の詳細

### 新機能

#### Claude Sonnet 4.6 のサポート追加

Claude Code が **Claude Sonnet 4.6** モデルに対応しました。最新モデルを利用することで、より高品質なコード生成や自然言語処理が期待できます。Claude Code の設定でモデルを指定する際に、新しいモデル ID `claude-sonnet-4-6` が使用可能になります。

#### スピナーヒントのカスタマイズ機能

処理中に表示されるスピナー（ローディングインジケーター）のヒントメッセージをカスタマイズできるようになりました。設定ファイルで以下のオプションが利用可能です。

- **`tips` 配列**: カスタムヒントメッセージを配列で指定できます
- **`excludeDefault: true`**: デフォルトのヒントを除外し、独自のヒントのみを表示できます

この機能により、チームや組織固有のヒントメッセージを表示したり、特定のワークフローに合わせた情報を提供したりすることが可能になります。

#### SDK 開発者向け: レート制限情報の新型追加

Claude Code の SDK を利用している開発者向けに、**`SDKRateLimitInfo`** と **`SDKRateLimitEvent`** の2つの新しい型が追加されました。これらの型を使用することで、以下の情報を含むレート制限ステータスのアップデートを受け取ることができます。

- 現在の利用率
- レート制限のリセット時間
- 制限超過に関する情報

この追加により、SDK を介して Claude Code を統合しているアプリケーションがレート制限をより適切にハンドリングできるようになります。

#### 追加ディレクトリからの設定読み込み

追加ディレクトリから **`enabledPlugins`** と **`extraKnownMarketplaces`** を読み込む機能が実装されました。これにより、プラグインやマーケットプレイスの設定をより柔軟に管理できるようになります。

### バグ修正

#### セッション圧縮時のスキル表示問題

会話のコンテキストが圧縮される際に、スキルが不正に表示されてしまう問題が修正されました。長時間のセッションでも正確なスキル情報が表示されるようになります。

#### `.claude.json.backup` ファイルの過剰蓄積の解消

設定ファイルのバックアップである `.claude.json.backup` が過剰に蓄積されてしまう問題が解決されました。ディスクスペースの無駄な消費を防ぐ修正です。

#### プラグインインストール後の再起動不要化

プラグインをインストールした後に Claude Code を再起動する必要があった問題が修正されました。プラグインのインストール後、即座に利用可能になります。

#### メモリ効率の改善

内部的なメモリ管理が改善され、長時間の利用時やリソースが限られた環境でのパフォーマンスが向上しています。

## 技術的なポイント

- **最新モデルへの対応**: `claude-sonnet-4-6` が利用可能になり、最新世代のモデルの性能を Claude Code から直接活用できます
- **カスタマイズ性の向上**: スピナーヒントの `tips` 配列と `excludeDefault` オプションにより、UI 表示を組織のニーズに合わせてカスタマイズできます
- **SDK の型安全性**: `SDKRateLimitInfo` と `SDKRateLimitEvent` 型の追加により、TypeScript を使用した SDK 統合でレート制限情報を型安全に扱えるようになります
- **プラグイン体験の改善**: インストール後の再起動不要化により、プラグインの導入・切り替えがスムーズになります
- **バックアップファイル管理**: `.claude.json.backup` の蓄積問題の解決により、長期運用時のディスク使用量が適切に管理されます

## まとめ

v2.1.45 は、最新モデルである Claude Sonnet 4.6 への対応を中心に、SDK 開発者向けの機能強化や UI カスタマイズ機能の追加が行われたリリースです。特に、プラグインインストール後の再起動不要化やメモリ効率の改善など、日常的な利用体験の向上に寄与する修正が多く含まれています。SDK を通じて Claude Code を統合している開発者にとっては、`SDKRateLimitInfo` と `SDKRateLimitEvent` 型の追加により、より堅牢なレート制限ハンドリングが実装できるようになります。継続的な改善が続く Claude Code から、今後の機能追加にも注目です。
