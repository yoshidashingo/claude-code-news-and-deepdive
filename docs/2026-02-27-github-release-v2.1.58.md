# Claude Code v2.1.58 リリース - Remote Control の対象ユーザー拡大

## 概要

Claude Code v2.1.58 では、**Remote Control（リモートコントロール）機能**の提供対象ユーザーが拡大されました。これまで限られたユーザーにのみ提供されていたこの機能が、より多くのユーザーに開放されています。

## 新機能

### Remote Control の対象拡大

Remote Control は、Claude Code をターミナル外部から操作・制御できる機能です。これにより、外部のツールやインターフェースから Claude Code のセッションに指示を送ったり、状態を監視したりすることが可能になります。

今回のリリースでは、段階的なロールアウトの一環として、この機能へのアクセスが可能なユーザー範囲が広げられました。ベータ機能として一部のユーザーに限定提供されていたものが、引き続き順次展開されています。

Remote Control が利用可能になると、以下のようなユースケースが広がります：

- **外部 UI からの操作**: Claude.ai などのウェブインターフェースから Claude Code セッションを遠隔操作する
- **自動化パイプラインとの連携**: CI/CD や外部スクリプトからプログラム的に Claude Code を制御する
- **マルチクライアント対応**: 複数の端末やクライアントから同一セッションへのアクセス

## 技術的なポイント

- **段階的なロールアウト**: Remote Control は機能フラグによる段階展開方式で提供されており、今回の変更はそのフラグの対象範囲を拡大するものです
- **アクセス確認**: Remote Control が自分のアカウントで有効になっているかどうかは、Claude Code の設定や claude.ai のアカウント画面から確認できます
- **既存ワークフローへの影響**: この機能はオプトイン形式であるため、利用しない場合は従来通りのターミナル操作に影響はありません

## まとめ

v2.1.58 は、Remote Control 機能の対象ユーザーを拡大するシンプルなリリースです。この機能が使えるようになることで、Claude Code の操作方法に新たな選択肢が生まれ、ターミナルに縛られない柔軟な開発ワークフローが実現できるようになります。まだアクセスできていないユーザーも、順次対象に含まれていく予定です。
